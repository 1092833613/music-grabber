# 📱 APK 安装与使用指南

**版本**: 1.0.0  
**适用**: Android 手机/平板

---

## 📥 第一步：下载 APK

### 1. 访问 Actions 页面

**https://github.com/1092833613/music-grabber/actions**

### 2. 找到成功的构建

1. 点击 **#4 运行**（应该显示绿色 ✅）
2. 滚动到页面**最底部**
3. 找到 **Artifacts** 区域

### 3. 下载文件

1. 点击 **`apk`** 链接
2. 保存文件到电脑
3. 文件名：`musicgrabber-1.0.0-debug.apk`
4. 大小：约 20-50 MB

---

## 📤 第二步：传输到手机

### 方法 1: USB 数据线（推荐）⭐

**Windows**:
1. USB 连接手机和电脑
2. 手机选择"文件传输"模式
3. 电脑打开"此电脑" → 找到你的手机
4. 复制 APK 到手机存储（如 Download 文件夹）

**Mac**:
1. USB 连接手机和电脑
2. 使用 Android File Transfer 工具
3. 复制 APK 到手机存储

---

### 方法 2: 微信/QQ 文件传输

**微信**:
1. 打开微信电脑版
2. 搜索"文件传输助手"
3. 拖拽 APK 到聊天窗口
4. 手机微信接收文件

**QQ**:
1. 打开 QQ 电脑版
2. 搜索"我的电脑"
3. 发送 APK 文件
4. 手机 QQ 接收文件

---

### 方法 3: 云盘

**百度网盘/Google Drive/OneDrive**:
1. 上传 APK 到云盘
2. 手机打开云盘 App
3. 下载 APK

---

### 方法 4: 蓝牙

1. 电脑蓝牙发送 APK
2. 手机蓝牙接收
3. 保存到存储

---

## 🔧 第三步：允许安装未知来源

### Android 8.0+（新系统）

**安装时会提示**，按以下步骤操作：

1. **点击 APK 安装**
2. **系统提示**："禁止安装未知应用"
3. **点击**："设置"
4. **开启**："允许来自此来源的应用"
5. **返回**，继续安装

### Android 7.0 及以下（旧系统）

1. **设置** → **安全**
2. **开启**："未知来源"
3. **确认**警告
4. **返回**，安装 APK

---

## 📲 第四步：安装 APK

### 安装步骤

1. **找到 APK 文件**
   - 文件管理器 → Download 或接收文件夹

2. **点击 APK**
   - 显示安装界面

3. **点击**："安装"
   - 等待安装进度

4. **完成**
   - 显示："应用已安装"
   - 点击："打开" 或 "完成"

## 🎵 第五步：使用音乐抓取 App

### 首次启动

1. **打开 App**
   - 图标：🎵 音乐抓取

2. **权限请求**

   └─────────────────────────┘

### 功能 1: 下载音频

**步骤**:

1. **复制链接**
   - 打开 YouTube/B 站/抖音
   - 复制视频链接

2. **粘贴到 App**
   - 粘贴到输入框

3. **选择格式**
   - MP3（推荐）
   - WAV（无损）
   - FLAC（高品质）
   - M4A（苹果设备）

4. **点击**："获取信息"
   - 显示标题、歌手、时长

5. **点击**："下载"
   - 等待下载完成
   - 保存到 `/storage/emulated/0/Download/music_grabber/`

---

### 功能 2: 搜索音乐 🆕

**步骤**:

1. **切换到**："搜索" 标签
2. **输入**：歌曲名或歌手
3. **选择音源**：
   - YouTube（推荐）
   - SoundCloud
   - Bandcamp
4. **点击**："🔍 搜索"
5. **选择结果**，直接下载

---

### 功能 3: 歌词下载 🆕

**自动下载**：
- 下载音频时自动匹配歌词
- 保存为 `.lrc` 文件
- 同名文件（如 `七里香.mp3` + `七里香.lrc`）

---

### 功能 4: 封面下载 🆕

**自动下载**：
- 下载音频时自动获取封面
- 嵌入到音频文件 ID3 标签
- 播放器会显示专辑封面

---

### 功能 5: 历史记录 🆕

**查看**：
1. 切换到："历史" 标签
2. 查看所有下载记录
3. 点击重新下载

**搜索**：
- 输入关键词搜索历史

**清空**：
- 点击："🗑️ 清空"

---

### 功能 6: 播放列表 🆕

**创建**：
1. 切换到："播放列表" 标签
2. 点击："➕ 新建播放列表"
3. 输入名称

**添加歌曲**：
- 下载时选择"添加到播放列表"

**管理**：
- 编辑、删除、导出播放列表

## 🎯 常见问题

### Q1: 安装失败

**错误**: "解析包时出现问题"

**解决**:
1. 检查 Android 版本（需要 Android 5.0+）
2. 重新下载 APK
3. 检查文件是否完整

---

### Q2: 无法打开

**错误**: "应用未安装" 或 "闪退"

**解决**:
1. 卸载后重新安装
2. 清除应用数据
3. 检查手机存储空间

---

### Q3: 下载失败

**错误**: "下载失败" 或 "网络错误"

**解决**:
1. 检查网络连接
2. 更换视频链接
3. 检查链接是否有效

---

### Q4: 找不到文件

**问题**: 下载后找不到文件

**解决**:
1. 打开文件管理器
2. 导航到：`Download/music_grabber/`
3. 按时间排序查找

---

### Q5: 没有声音

**问题**: 播放时没有声音

**解决**:
1. 检查手机音量
2. 更换播放器
3. 重新下载（可能文件损坏）
