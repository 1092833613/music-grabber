## ⚠️ 异常情况处理

### 如果显示失败（红色 ❌）

**步骤**:

1. **查看详细日志**:
   ```
   Actions → 点击失败的构建 → 查看每个步骤
   ```

2. **常见错误**:
   - `Connection timeout` → 网络问题
   - `Module not found` → 依赖问题
   - `Build failed` → 编译错误

3. **重新运行**:
   ```
   Actions → Run workflow → 选择 main 分支 → 运行
   ```

### 如果超过 70 分钟还在运行

**可能原因**:
- 网络慢（下载 Android SDK）
- 编译卡住
- 资源不足

**解决方法**:
1. 取消当前运行
2. 检查日志找出卡住的步骤
3. 重新运行工作流
