# 应用图标说明

## 📱 当前状态

**当前使用**: Kivy 默认图标

## 🎨 建议图标设计

### 尺寸要求

- **主图标**: 512x512 PNG
- **自适应图标**: 1080x1080 PNG（Android 8.0+）
- **通知栏图标**: 24x24 PNG（单色）

### 设计建议

**主题**: 音乐 + 下载

**元素**:
- 🎵 音符符号
- ⬇️ 下载箭头
- 渐变背景（蓝色/紫色）

**风格**:
- 简洁现代
- 高对比度
- 易于识别

### 在线工具

**免费图标生成**:
- https://www.canva.com/
- https://icon.kitchen/
- https://romannurik.github.io/AndroidAssetStudio/

### 使用方法

1. 设计 512x512 PNG 图标
2. 命名为 `icon.png`
3. 放到项目根目录
4. 在 `buildozer.spec` 中配置：
   ```ini
   icon.filename = icon.png
   ```
5. 重新打包
